  <action>
    Create `scripts/ingest_documents.py` (per D-14) following the established script pattern from `run_sift.py`. The module structure:

    **Module docstring:**
    ```python
    """Document ingestion pipeline for cross-domain knowledge graph framework.

    Discovers, parses, and triages documents from a local corpus directory.
    Produces per-document text files and a triage.json metadata report.

    Usage:
        python ingest_documents.py /path/to/corpus --output ./output --domain contract
        python ingest_documents.py /path/to/corpus --output ./output --triage-only
    """
    ```

    **Imports:** `from __future__ import annotations`, `json`, `re`, `sys`, `datetime` (timezone, datetime), `pathlib.Path`. Wrap sift-kg reader in try/except:
    ```python
    try:
        from sift_kg.ingest.reader import read_document
        HAS_SIFT_READER = True
    except ImportError:
        HAS_SIFT_READER = False
    ```

    **Constants:**
    ```python
    KNOWN_CATEGORIES = {"hotel", "pcc", "av", "catering", "security", "ems"}
    SUPPORTED_EXTENSIONS = {".pdf", ".xls", ".xlsx", ".eml", ".doc", ".docx", ".txt", ".html", ".htm"}
    ```

    **Functions (all with full type hints and docstrings):**

    1. `discover_corpus(corpus_path: Path) -> list[Path]` — (per D-11) Recursively walk `corpus_path`, collect files with extensions in `SUPPORTED_EXTENSIONS`. Return sorted list of Path objects. Use `corpus_path.rglob("*")`, filter `p.is_file()` and `p.suffix.lower() in SUPPORTED_EXTENSIONS`.

    2. `sanitize_doc_id(filename: str) -> str` — (per D-13) Strip extension, lowercase, replace non-alphanumeric chars (except underscore) with underscore using `re.sub(r"[^a-z0-9_]", "_", name.lower())`. Strip leading/trailing underscores.

    3. `detect_category(file_path: Path, corpus_root: Path) -> str` — (per D-10) Get the relative path from corpus_root, take first part (top-level folder). If lowercase matches a value in `KNOWN_CATEGORIES`, return it lowercase. Otherwise return `"uncategorized"`. If file is directly in corpus_root (no subfolder), return `"uncategorized"`.

    4. `parse_document(file_path: Path) -> str | dict` — (per D-04, D-05, D-06) Use `read_document(file_path)` from sift-kg. On success, return the text string. On any exception (FileNotFoundError, import error, parse error), return `{"error": str(e), "file": str(file_path)}` dict. If `HAS_SIFT_READER` is False, try a plain text fallback for .txt files, otherwise return error dict.

    5. `compute_readiness_score(text: str, file_size: int) -> float` — Heuristic score 0.0-1.0 based on: text density (text_length / file_size ratio, capped at 1.0, weight 0.4), structure signals (presence of section headers/numbered clauses via regex, weight 0.3), minimum length threshold (>500 chars = 0.3, >100 chars = 0.15, else 0.0). Return weighted sum rounded to 2 decimal places.

    6. `classify_parse_type(text: str, file_size: int) -> str` — (per D-04) Return `"text"` if text density (len/file_size) > 0.1, `"scanned"` if text density < 0.01, `"mixed"` otherwise. These thresholds are heuristics: text-native PDFs have high text/byte ratios, scanned PDFs have very low ratios after OCR.

    7. `build_document_metadata(file_path: Path, corpus_root: Path, text: str | dict, doc_id: str) -> dict` — (per D-08) Build and return the per-document metadata dict:
       ```python
       {
           "doc_id": doc_id,
           "filename": file_path.name,
           "file_path": str(file_path),
           "file_size_bytes": file_path.stat().st_size,
           "page_count": None,  # available for PDFs if parseable
           "category": detect_category(file_path, corpus_root),
           "parse_type": classify_parse_type(text, file_size) if isinstance(text, str) else "failed",
           "text_length": len(text) if isinstance(text, str) else 0,
           "parse_errors": text.get("error", None) if isinstance(text, dict) else None,
           "extraction_readiness_score": compute_readiness_score(text, file_size) if isinstance(text, str) else 0.0,
       }
       ```

    8. `ingest_corpus(corpus_path: Path, output_dir: Path, domain_name: str | None = None) -> dict` — (per D-07, D-12, D-15) Main orchestrator:
       - Validate corpus_path exists
       - Create `output_dir / "ingested"` directory
       - Call `discover_corpus(corpus_path)`
       - Use `rich.progress.Progress` for progress bar (per D-15): `with Progress() as progress: task = progress.add_task("Ingesting...", total=len(files))`
       - For each file: `sanitize_doc_id`, `parse_document`, `build_document_metadata`
       - If parse succeeded (text is str): write text to `output_dir / "ingested" / f"{doc_id}.txt"` (per D-12)
       - If parse failed (text is dict): log error, increment fail count (per D-05, D-08)
       - Collect all metadata dicts into a list
       - Write `triage.json` to output_dir: `{"ingested_at": ISO timestamp, "corpus_path": str, "domain": domain_name, "total_files": N, "successful": N, "failed": N, "documents": [metadata_list]}` (per D-07, D-09)
       - Print summary to stdout
       - Return the triage dict

    **`__main__` block** (per D-14, following run_sift.py pattern):
    ```python
    if __name__ == "__main__":
        if len(sys.argv) < 2:
            print(__doc__)
            sys.exit(1)

        corpus = Path(sys.argv[1])
        output = Path(sys.argv[sys.argv.index("--output") + 1]) if "--output" in sys.argv else Path("./epistract-output")
        domain = sys.argv[sys.argv.index("--domain") + 1] if "--domain" in sys.argv else None

        result = ingest_corpus(corpus, output, domain)
        print(json.dumps({"status": "complete", "total": result["total_files"], "successful": result["successful"], "failed": result["failed"]}, indent=2))
    ```
