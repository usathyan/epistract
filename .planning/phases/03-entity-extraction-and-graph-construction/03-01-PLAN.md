Create `scripts/chunk_document.py` per D-01, D-02, D-03:

1. Module docstring with usage: `python chunk_document.py <doc_id> <output_dir> [--max-size 10000]`

2. Constants:
   - `MAX_CHUNK_SIZE = 10000`
   - `MIN_CHUNK_SIZE = 500`
   - `SECTION_PATTERNS`: list of compiled regexes ordered by specificity:
     - `r"^(ARTICLE\s+[IVXLCDM\d]+[.:]\s*.*)$"` (IGNORECASE, MULTILINE)
     - `r"^(Section\s+\d+[\.\d]*[.:]\s*.*)$"` (IGNORECASE, MULTILINE)
     - `r"^(\d+\.\d+[\.\d]*\s+[A-Z].*)$"` (MULTILINE)
     - `r"^([A-Z][A-Z\s]{3,}:?\s*)$"` (MULTILINE) for ALL CAPS headers

3. `chunk_document(text: str, doc_id: str, max_size: int = MAX_CHUNK_SIZE) -> list[dict]`:
   - Try `_split_at_sections(text)` first
   - If produces > 1 section: merge small adjacent sections below MIN_CHUNK_SIZE via `_merge_small_sections(sections, doc_id, max_size)`
   - If only 1 section (no clause structure detected): fallback to `_split_fixed(text, doc_id, max_size)` (per D-02)
   - Return list of dicts: `{"chunk_id": f"{doc_id}_chunk_{i:03d}", "text": str, "section_header": str, "char_offset": int}`

4. `_split_at_sections(text: str) -> list[tuple[str, str, int]]`:
   - Find all matches for any SECTION_PATTERN
   - Split text at those positions
   - Return list of (header, body_text, char_offset) tuples
   - Include preamble (text before first header) as a section with header=""

5. `_merge_small_sections(sections, doc_id, max_size)`:
   - Walk sections, merge consecutive sections if combined text < max_size
   - Oversized sections (> max_size): split at paragraph boundaries (double newline)
   - Return list of chunk dicts

6. `_split_fixed(text: str, doc_id: str, max_size: int) -> list[dict]`:
   - Split at paragraph boundaries (double newline `\n\n`)
   - Accumulate paragraphs up to max_size
   - Return chunk dicts with section_header=""

7. `chunk_document_to_files(doc_id: str, output_dir: Path, max_size: int = MAX_CHUNK_SIZE) -> Path`:
   - Read `{output_dir}/ingested/{doc_id}.txt`
   - Call `chunk_document()`
   - Write each chunk as JSON to `{output_dir}/ingested/{doc_id}_chunks/chunk_NNN.json` (per D-03)
   - Return the chunks directory path

8. CLI `if __name__ == "__main__"`:
   - Parse sys.argv: `<doc_id> <output_dir> [--max-size N]`
   - Call `chunk_document_to_files()`
   - Print JSON summary: `{"doc_id": str, "chunks": int, "chunk_dir": str}`

Follow project conventions: pathlib, `from __future__ import annotations`, json.dumps indent=2, no argparse.
