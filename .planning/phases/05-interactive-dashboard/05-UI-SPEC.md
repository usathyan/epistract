## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-text gaps, inline badge padding |
| sm | 8px | Compact element spacing, input inner padding vertical |
| md | 16px | Default element spacing, card padding, input inner padding horizontal |
| lg | 24px | Section padding, panel headers |
| xl | 32px | Layout gaps between major panels |
| 2xl | 48px | Chat message group vertical spacing |
| 3xl | 64px | Not used in this phase |

Exceptions: Chat input area fixed height 56px (not a spacing token, a component dimension). Sidebar width 280px collapsed to 0px. Side panels width 420px.

## Typography

| Role | Size | Weight | Line Height | Font |
|------|------|--------|-------------|------|
| Body | 15px | 400 (regular) | 1.5 | system sans-serif |
| Label | 13px | 600 (semibold) | 1.4 | system sans-serif |
| Heading | 20px | 600 (semibold) | 1.2 | system sans-serif |
| Data | 14px | 400 (regular) | 1.4 | "SF Mono", "Fira Code", "Consolas", monospace |

Notes:
- Body (15px) used for chat messages, source document text, node detail descriptions
- Label (13px) used for entity type badges, filter labels, panel tab labels, timestamps
- Heading (20px) used for panel titles ("Graph", "Sources"), section headings in chat responses
- Data (14px monospace) used for cost figures, dates, clause references, contract IDs (per D-25 "monospace for data")
- Markdown rendering in chat responses uses Body size; tables rendered in Data font at 13px

## Layout Contract

### Three-Column Structure (D-05)

```
+------------------+----------------------------+--------------------+
|    Sidebar       |      Chat Panel            |   Side Panel       |
|   (280px)        |     (flex: 1)              |   (420px)          |
|                  |                            |                    |
| Logo/title       | Chat messages              | Graph OR Sources   |
| Nav links:       | (scrollable)               | (tabbed)           |
|  - Chat          |                            |                    |
|  - Graph         |                            | Graph: vis.js      |
|  - Sources       |                            |   + search bar     |
|                  |                            |   + type toggles   |
|                  |                            |   + severity filter|
| Severity legend  |                            |                    |
| Entity type      |                            | Sources: text view |
| legend           |                            |   + section scroll |
|                  |                            |   + "Open PDF" link|
|                  +----------------------------+                    |
|                  | Chat input bar (fixed)     |                    |
+------------------+----------------------------+--------------------+
```

- Sidebar: Fixed left, 280px wide, `#1a1a2e` background, white text
- Chat Panel: Fluid center column, min-width 480px
- Side Panel: Collapsible right, 420px wide, toggled by sidebar nav or citation clicks
- Side Panel starts collapsed on load; chat is full-width initially
- Clicking "Graph" or "Sources" in sidebar opens the corresponding side panel tab

### Responsive Behavior

- Below 1024px viewport: sidebar collapses to hamburger icon (48px wide), side panel becomes overlay
- Below 768px: side panel becomes full-screen overlay with close button
- Chat panel always visible as primary interface

## Interaction Contract

### Chat Interface (D-09, D-10, D-11, D-12)

| Interaction | Behavior |
|-------------|----------|
| First load | Display 4-6 starter prompt cards in chat area, no messages yet |
| Click starter prompt | Populate input with prompt text, auto-send |
| Send message | POST to `/api/chat`, show user bubble, stream assistant response |
| Streaming response | Render markdown incrementally (tables, lists, citations) |
| Citation click `[Contract Name]` | Open Sources panel, scroll to referenced section, highlight |
| Esc from Sources | Close Sources panel, return focus to chat input |
| Empty input submit | No-op, do not send |
| API error | Show inline error message below last user message |

### Graph Panel (D-16, D-17)

| Interaction | Behavior |
|-------------|----------|
| Open graph panel | Render vis.js network from graph_data.json |
| Search bar | Filter nodes by name substring, highlight matches |
| Entity type toggles | Toggle visibility of node types (buttons with entity colors) |
| Severity filter | Dropdown: All / Critical / Warning / Info |
| Click node | Show node detail popover (type, attributes, source docs, connections) |
| Double-click node | Recenter graph on node neighborhood (D-17) |
| "Ask about this" button | Populate chat input with "Tell me about [node name]" and focus chat |

### Source Panel (D-18, D-19)

| Interaction | Behavior |
|-------------|----------|
| Open sources panel | Show list of ingested documents |
| Select document | Load ingested text from `/api/sources/{doc_id}` |
| Citation navigation | Scroll to referenced section, highlight with `#fffbeb` background |
| "Open original PDF" | Link opens PDF in new browser tab via `/api/pdf/{doc_id}` |
| Search within source | Ctrl+F browser native search (no custom search UI needed) |

## Component Inventory

Since this is vanilla HTML/JS (no component library), these are the custom elements to implement:

| Component | Element | Notes |
|-----------|---------|-------|
| ChatMessage | `<div class="chat-msg chat-msg--user\|assistant">` | User messages right-aligned blue bg, assistant left-aligned white bg |
| StarterCard | `<button class="starter-card">` | Rounded card with question text, accent border on hover |
| SidePanel | `<aside class="side-panel side-panel--open\|closed">` | Animated slide-in from right, 420px |
| NodePopover | `<div class="node-popover">` | Positioned near clicked node, shows attributes |
| SeverityBadge | `<span class="badge badge--critical\|warning\|info">` | Pill-shaped, colored per severity palette |
| EntityBadge | `<span class="entity-badge entity-badge--{type}">` | Small colored dot + type label |
| ToggleButton | `<button class="toggle-btn toggle-btn--active">` | Entity type filter, colored indicator |
| ChatInput | `<form class="chat-input">` | Fixed bottom bar, textarea + send button |
| SourceViewer | `<div class="source-viewer">` | Scrollable text with highlighted sections |
| DocList | `<ul class="doc-list">` | Sidebar-style document list in Sources panel |
