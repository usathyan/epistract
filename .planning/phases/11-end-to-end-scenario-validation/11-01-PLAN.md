  <action>
    1. Create `tests/regression/run_regression.py` — the main regression runner (per D-09):
    - Uses `argparse` with these flags:
      - `--baselines DIR` — path to baseline directory (default: `tests/baselines/v2/`, fallback to `tests/baselines/v1/`)
      - `--output-dir DIR` — path to scenario output directories (default: `tests/corpora/`)
      - `--skip-extraction` — only validate existing output, don't run extraction (for quick checks)
      - `--update-baselines` — save current output as new V2 baselines
      - `--scenario NAME` — run only a specific scenario (e.g., `s01_picalm`)
    - Main flow:
      a. Load all baseline JSON files from `--baselines` dir using `glob("*.json")`
      b. For each baseline, locate the corresponding output directory:
         - Drug discovery: `tests/corpora/{scenario}/output/`
         - Contracts: check `sample-output/` (skip with warning if not found, per pitfall 2)
      c. Call `load_graph_data()` and `load_epistemic_data()` for each scenario
      d. Call `compare_scenario()` for each
      e. Call `format_report()` to print results
      f. Exit 0 if all pass, exit 1 if any fail
    - `--update-baselines` mode: For each scenario, read current output, create baseline JSON in `tests/baselines/v2/`, print what was updated
    - `--skip-extraction` mode: Same as default (the runner doesn't do extraction anyway — extraction happens via plugin per D-04). This flag is for documentation/intent clarity.
    - Use `sys.exit(0)` on success, `sys.exit(1)` on failure
    - Print colored output using simple ANSI codes (green PASS, red FAIL) — no Rich dependency needed
    - Include a `def main()` entry point and `if __name__ == "__main__": main()`

    2. Add to `Makefile` (append after existing `clean` target):
    ```makefile

    regression: ## Run regression suite against baselines
    	python tests/regression/run_regression.py

    regression-update: ## Update V2 baselines from current output
    	python tests/regression/run_regression.py --update-baselines

    regression-check: ## Quick validation of existing output (no extraction)
    	python tests/regression/run_regression.py --skip-extraction
    ```

    Use tab indentation for Makefile recipes (not spaces). Preserve all existing Makefile content exactly as-is.
  </action>
